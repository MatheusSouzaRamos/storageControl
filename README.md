# storageControl

Sistema de gerenciamento de estoque desenvolvido com backend em Java com o framework Spring Boot e HTML, CSS e JavaScript no frontend.

## Funcionalidades

- Gerenciamento Completo: Cadastro, edição, alteração direta na quantidade e exclusão de produtos em uma aba de configurações.
- Ajuste Rápido: Botões na tela principal para somar ou subtrair uma unidade do estoque com um clique.
- Busca Prática: Filtro por ID ou Nome na página principal.

Crie um banco PostgreSQL:


### Executar o frontend
